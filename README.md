# David Bowie

Another discord bot aimed to be the jack of all trades but master of none

# Running

```bash
# Clone this repo
git clone https://github.com/scinscinscin/DavidBowie.git
cd DavidBowie

# Compile to javascript
tsc

# Populate the token.json file in the config folder
mv token_example.json token.json
vim token.json

# Run
yarn start
```
